Running in Visual Studio Code

Using Live Server
Direct Opening
  1. Install Live Server Extension:
    • Click the Extensions icon in VS Code (or press Ctrl+Shift+X)
    • Search for "Live Server" by Ritwick Dey
    • Click the Install button
  2. Open your project:
    • Open the folder containing your HTML file in VS Code (File > Open Folder)
    • Ensure your HTML file is named index.html
  3. Start Live Server:
    • Right-click on your HTML file in the Explorer pane
    • Select "Open with Live Server"
    • Your browser will open automatically with the address http://127.0.0.1:5500/index.html
  4. Benefits of Live Server:
    • Auto-reload when you make changes to your code
    • Simulates a real web server environment
    • Better compatibility with certain web features
Note: If you see a "Port 5500 is already in use" message, Live Server will automatically try the next available port (5501, 5502, etc.).
  1. Save your HTML file:
    • Create a new file in VS Code (File > New File)
    • Copy and paste the HTML code
    • Save the file as index.html (Ctrl+S)
  2. Open in browser:
    • Right-click on the HTML file in VS Code's file explorer
    • Select "Reveal in File Explorer" (Windows) or "Reveal in Finder" (Mac)
    • Double-click the HTML file to open it in your default browser
  3. Alternative method:
    • You can also drag the HTML file from VS Code directly into your browser window
    • Or use the VS Code command palette (Ctrl+Shift+P) and search for "View: Show Preview"
Note: When opening HTML files directly in the browser, some features may not work due to browser security restrictions (like loading external resources). Live Server is recommended for full functionality.
